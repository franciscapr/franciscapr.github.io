About Me

Soy una desarrolladora de software viñamarina de 29 años. Mi pasión por la programación comenzó con mi tesis profesional, donde descubrí el lenguaje R. Tras recibir becas para emprender, participé en un programa de Talento Digital sobre Ruby on Rails. Recientemente, obtuve una beca del Banco de Chile para aprender Python con Flask y ahora estoy ampliando mis conocimientos integrando Django en mis proyectos.

Entrenamiento K-NN

Se utilizaron herramientas como Python, Numpy, Pandas, Sklearn, Jupyter y datos en crudo extraidos desde Kaggle para el entrenamiento de un modelo de clasificación para la predicción de pacientes con antecedentes familiares de sobrepeso.

EasyBay

Se utilizaron herramientas para desarrollar una apliación de tipo front-end como HTML, CSS, JS y Bootstrap para dar estilo y funcionalidad a la página web EasyBay.
